## 📌 Overview

Farmer Connect UI (Vivasai) is an AI-powered web application built to support farmers in Tamil Nadu by providing crop predictions, risk alerts, market prices, and government scheme information — all in one place.

The platform aims to reduce the information gap in agriculture by delivering essential insights through a simple and user-friendly interface. It helps farmers make better decisions related to crop selection, pricing, and accessing government benefits.

By focusing on accessibility and clarity, the application ensures usability even for users with minimal technical experience.

---

## ✨ Features

- 🌾 **Built for Tamil Nadu Farmers**  
  Designed specifically for regional farmers with a focus on local relevance.

- 🤖 **AI-Powered Crop Prediction**  
  Provides intelligent crop suggestions to support better planning.

- ⚠️ **Risk Alerts & Insights**  
  Notifies users about potential risks affecting crops.

- 📊 **Daily Market Prices**  
  Helps farmers track current prices and make informed selling decisions.

- 🏛️ **Government Schemes Information**  
  Centralized access to schemes and benefits.

- 🧑‍🌾 **All-in-One Platform**  
  Combines multiple essential tools in a single interface.

- 📱 **Responsive Design**  
  Works across mobile, tablet, and desktop.

- 🎨 **Clean & Intuitive UI**  
  Simple navigation for better usability.
